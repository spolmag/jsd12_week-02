## Problem Statement
    # หมู่บ้านในต่างจังหวัดที่ผมอยู่ปัจจุบัน เป็นหมู่บ้านขนาดค่อนข้างใหญ่ ประมาณ 700-1000 ครัวเรือน มีร้านขายของชำขนาดเล็กอยู่ 3 ร้าน
    # หมู่บ้านห่างจากตัวอำเภอที่มี 7-11, Lotus Express ประมาณ 6 กม. เกินระยะส่งของของ Seven Delivery การเดินทางไปตัวอำเภอต้องใช้มอเตอร์ไซค์หรือรถยนต์ ไม่สะดวกสำหรับบางครอบครัว และอันตรายสำหรับแม่บ้าน, เด็ก
    # ร้านของชำในหมู่บ้าน ขายสินค้าเครื่องอุปโภค-บริโภค แต่มักจะมีสินค้าไม่ครบ หรือยี่ห้อไม่ตรงกับที่ต้องการ หลายครั้งที่ลูกค้าไปที่ร้านแต่ไม่ได้ของที่ต้องการ เนื่องจากร้านไม่มีสต็อก
## Solutions
    # สร้าง Web Application ให้คนในหมู่บ้านสามารถเข้ามาเช็ครายการสินค้า -> หยิบลงตระกร้า -> ทำรายการซื้อ -> เลือกวิธีการรับสินค้า (รับที่หน้าร้าน, ให้ส่งให้ที่บ้าน) -> เลือกวิธีชำระเงิน -> สั่งซื้อ
